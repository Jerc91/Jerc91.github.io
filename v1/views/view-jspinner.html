JSpinner

Éste plugin es para crear animaciones representando un spinner; así mismo representa la espera por parte del usuario ya que la página esta realizando una acción. Con ello se superpone un elemento con el spinner en su interior para bloquear los controles de abajo.

Cuando pases el mouse por encima de cada uno de los spinners se mostrará el nombre del tema, que va desde el 1 hasta el 14 y se mostrará el ancho y alto mínimo de cada uno de los spinners.

Las animaciones fueron copiadas de las páginas: projects.lukehaas.me/css-loaders, tobiasahlin.com/spinkit. Estas animaciones se pasan a archivos .less para ajustarlas al plugin JSpinner.

  • API

  • fnBlockUI()

    Función para agregar un spinner al elemento para simular un bloqueo sobre el mismo, se agrega una animación que representa un spinenr. El siguiente código muestra como podemos bloquear parcialmente un elemento.

    Para el siguiente ejemplo se agregará un spinner aleatorio a un elemento div.

    Selecciona un tema!

      

    var btnSpinnerShow = document.querySelector('#btnSpinnerAShow');

btnSpinnerShow.addEventListener('click', fnSpinner, false);

//---------------------------------
// Función para mostrar o bloquear elementos
function fnSpinner() {
    j.jspinner.fnBlockUI({ query: '#demoSpinnerA', relative: true });
} // end function
//---------------------------------

  • fnUnBlockUI()

    Oculta la animación que bloquea la interfaz.

    Para el siguiente ejemplo se agregará un spinner aleatorio a un elemento div, dentro de las acciones que se tienen para éste ejemplo es: mostrar el spinner y eliminar el mismo.

    Selecciona un tema!

      

    var btnSpinnerShow = document.querySelector('#btnSpinnerBShow');
var btnSpinnerFade = document.querySelector('#btnSpinnerBFade');

btnSpinnerShow.addEventListener('click', fnSpinner, false);
btnSpinnerFade.addEventListener('click', fnSpinner, false);

//---------------------------------
// Función para mostrar o bloquear elementos
function fnSpinner() {
    if(this.id == 'btnSpinnerBShow') {
        j.jspinner.fnBlockUI({ query: '#demoSpinnerB', relative: true });
    } else {
        j.jspinner.fnUnBlockUI({ query: '#demoSpinnerB', relative: true });
    } // end if
} // end function
//---------------------------------
