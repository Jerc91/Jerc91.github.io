JLightbox

Plugin para generar un lightbox, con esto podemos mostrar información como ayuda, velo por tu mismo con el ejemplo de arriba un juego de luces.

Algunos colores se han tomado de la sigueinte página: uigradients.com

Enseguida se muestran los eventos que tiene el JSlide:

  • API

  • fnInit()

    Función para cargar los elementos del lightbox en un elemento especifico, o para cargarlo en el body, el parámetro que recibe la función es el selector del elemento contenedor del lightbox, éste parámetro es opcional, de no pasarse nada el elemento contenedor será el body.

    El siguiente ejemplo muestra como configurar el inicio del lightbox:

    j.jlightbox.fnInit();
console.log(document.querySelector('#lightbox'))

  • fnShow()

    Función para mostrar el lightbox, como parámetros recibe el HTML que se agregará en su interior, como parámetro opcional se recibe verdadero o false, para mantener el lightbox hasta que se cierre con el API, éste permanecerá de forma permanente, no se ocultará ni con el botón de X, ni con la tecla ESC.

    El siguiente ejemplo muestra como funciona el lightbox:

    j.jlightbox.fnInit();
// Utilizo el código del < para mostrar el código.
j.jlightbox.fnShow('<h4>!Hola Mundo¡</h4>', true);

  • fnClose()

    Función para ocultar el lightbox, no recibe parámetros..

    El siguiente ejemplo muestra como ocultar el lightbox:

    j.jlightbox.fnInit();
j.jlightbox.fnShow('<h4>!Hola Mundo¡</h4>', true);
// Se cerrará el lightbox pasado a 6 segundos (6000)
setTimeout(function() {
	j.jlightbox.fnClose(close);
}, 6000);
