JSlide

    Plugin para generar un slide con la posibilidad de mostrar uno o varios items por slide, para su uso se debe contar con el plugin JMain. Se deberá usar un elemento contenedor del JSlide junto con una lista no ordenada que será el JSlide, cada li es un item que podrá contener un slide.

    La instancia del plugin recibe como parámetro un objeto con los siguientes atributos:

    // Objeto para inicializar el JSlide
var configJSlide = {
	buttons: true,
    fnCall: function() { } // end fnCall
    list: 9,
    maxElementsShow: 3,
    name: 'main',
    qSlide: ".jslide",
    qWrapper: '.wrapperMovies',
    width: 600,
    withTouch: true
}; // End object Slide
    PropiedadProposito
    buttonsSirve para asignar funciones a los eventos clic de los botones que deben estar enseguida del contenedor del JSlide, para identificar los botones estos deben tener las clases: btnLeft y btnRight, para el botón de la izquierda y para la derecha respectivamente.
    fnCallFunción de callback, esta se ejecutará una vez el JSlide se ha creado.
    listIndica el total de items del JSlide, éste campo puede ser númerico o un arreglo.
    maxElementsShowIndica el total de items a mostrar por cada slide en el JSlide.
    nameLas animaciones del JSlide son creadas por una hoja de estilos en tiempo real, que se crea cuando se inicia éste, esta se referencia con una etiquieta link. El "name" sirve para eliminar la etiqueta una vez se necesite reemplezar o eliminar la tag.
    qSlideEs el selector que identifica al JSlide para ser configurado.
    qWrapperEs el selector que identifica al contenedor del JSlide, y al contenedor de los botones con las clases: btnLeft y btnRight.
    widthAncho que se establece para el JSlide, el contenedor de éste puede tener diferente tamaño, la propiedad solo afecta al JSlide.
    withTouchCuando se usa en un dispositivo con touch, esta propiedad brinda la posibilidad de cambiar de slides con los eventos de touch.

    Enseguida se muestran los eventos que tiene el JSlide:

    • API

    • fnGetInstance()

      Función para obtener la instancia del JSlide, es el único método que tiene el espacio de nombres j.jslide, los métodos que contiene el plugin se usarán en la instancia.

      El siguiente ejemplo muestra como usar la instancia.

      var jslide = j.jslide.fnGetInstance();
console.log(jslide);

    • fnInit()

      Función para generar el JSlide, la función recibe el objeto explicado al principio de esta página.
