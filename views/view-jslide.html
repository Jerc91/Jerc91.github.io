JSlide

    Genera un slide con la posibilidad de mostrar uno o varios items por slide. Se deberá usar un elemento contenedor del JSlide junto con una lista no ordenada que será el JSlide, cada li es un item que podrá contener un slide.

    Esté plugin esta basado en estilos dinámicos que con el ancho del contenedor se calculas las animaciones, para usar la propiedad transform translate, y así usar las transcisiones de CSS3 y evitar un reflow cada vez que se cambie de slide.

    La instancia del plugin recibe como parámetro un objeto con los siguientes atributos, en donde todos excepto qSlide son opcionales:

    {
	qSlide: '',
	qWrapper: '',
	leftToRight: true
	maxElementsShow: 1,
	withTouch: false,
	timingFunction: 'cubic-bezier(.4, 0, .2, 1)',
	speedAnimation: 250,
	buttons: true,
	maxWidth: 0,
	width: ''
};
    PropiedadProposito
    qSlideSelector que identifica al elemento ui.JSlide para ser configurado.
    qWrapperSelector que identifica al contenedor del JSlide, y al contenedor de los botones con las clases: btnLeft y btnRight.
    leftToRightIndica si el movimiento se va hacer de izquierda a derecha, o por el contrario de derecha a izquierda.
    maxElementsShowIndica el total de items a mostrar por cada slide en el JSlide.
    withTouchCuando se usa en un dispositivo con touch, esta propiedad brinda la posibilidad de cambiar de slides con el touch.
    timingFunctionInidica la función con la que se realiza la transición.
    speedAnimationIndica los milisegundos que tardará la transición.
    buttonsIndica si se asigna el comportamiento de sigiuente y anterior a elementos con clase btnLeft, y btnRight.
    maxWidthIndica el ancho máximo.
    widthAncho que se establece para el JSlide, el contenedor de éste puede tener diferente tamaño, la propiedad solo afecta al JSlide.
    nameLas animaciones del JSlide son creadas por una hoja de estilos en tiempo real, que se crea cuando se inicia éste, esta se referencia con una etiquieta link. El atributo "data-name" sirve para eliminar la etiqueta una vez se necesite reemplezar o eliminar la tag.
