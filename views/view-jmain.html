JMain

Se encarga de almacenar las respuestas del servidor mediante de Service Worker y CacheStorage o, Web Workers y FileSystem (solo para Chrome), así el archivo service-worker.js es el encargando de usar el service worker y el CacheStorage, mientras service-worker.js se encarga de usar el API de FileSytem, y por último tenemos funciones comunes que usan tanto los Web Workers como el Service Worker.

El uso de Service Worker y los Web Workers ayudan a no bloquear la ejecución principal del motor de JavaScript, aprovechando así la ejecución paralela se realizan las peticiones de manera simultánea hacia el servidor.

Además de las peticiones asíncronas se tiene utilidades para crear espacios de nombres, junto con algunos patrones de diseño comunes y el uso de Service Locator, y métodos de ayuda.
