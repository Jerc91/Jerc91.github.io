JMain

Plugin para crear espacios de nombres en JavaScript de una forma facil. Se pueden cargar archivos en forma asíncrona mediante el uso de WebWorkers y XMLHTTPRequest.

El uso de los WebWorkers permite crear hilos que serán ejecutados al mismo tiempo, y así poder hacer peticiones simultaneas al servidor, con ello se puede ejecutar código principal mientras el servidor entrega una respuesta a nuestros WebSorkers.

Este plugin esta acompañado por otro archivo JavaScrtipt worker-import.js, El cual usa el API FileSystem en cada WebWorker obtenido se guarda el resultado de la petición si así se decide.

Dentro de éste plugin se encuentran métodos para arreglos, nodos HTML, espacio de nombres, peticiones asíncronas y herramientas.
