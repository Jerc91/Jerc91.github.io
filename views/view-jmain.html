JMain

Plugin para crear espacios de nombres en JavaScript de manera fácil. Se pueden cargar archivos en forma asíncrona mediante el uso de WebWorkers, XMLHttpRequest.

El uso de los WebWorkers permite crear hilos que serán ejecutados al tiempo con la aplicación, y así hacer peticiones simultaneas al servidor, estos seran los encargados de adminitrar las peticiones y las respuestas hacia y desde el servidor.

Este plugin esta acompañado por otros archivos JavaScrtipt worker-import.js, service-worker.js, worker-tools.js, fetch (polyfill). Los cuales usan las interfaces de FileSystem(Solo para Chrome), Promise y Fetch, cada WebWorker obtiene la respuesta del servidor y esta se guarda en el navegador de ser configurada la petición para ser guardada, esto funciona igual para el service worker con la diferencia que se usa el cache storage.

Dentro de éste plugin se encuentran métodos de arreglos, herramientas, espacio de nombres, peticiones asíncronas y patrones de diseño.
