JSpinner

Crea animaciones representando un spinner; así mismo representa la espera mientras se termina una tarea. Con ello se sobrepone un elemento con el spinner en su interior para bloquear los controles que oculta.

Cuando pases el mouse por encima de cada uno de los spinners se mostrará el nombre del tema, que va desde el 1 hasta el 14 y se mostrará el ancho y alto mínimo de cada uno.

Las animaciones fueron copiadas de las páginas: projects.lukehaas.me/css-loaders, tobiasahlin.com/spinkit. Estas animaciones se pasan a archivos .less para ajustarlas al plugin JSpinner.

  • API

  • jr.jspinner.block()

    Función para agregar un spinner al elemento para simular un bloqueo sobre el mismo, se agrega una animación que representa un spinner. El siguiente código muestra como podemos bloquear parcialmente un elemento.

    Para el siguiente ejemplo se agrega un spinner aleatorio a un elemento div.

    Selecciona un tema!

      

    var btnSpinnerShow = document.querySelector('#btnSpinnerAShow');

btnSpinnerShow.addEventListener('click', fnSpinner, false);

//---------------------------------
// Función para mostrar o bloquear elementos
function fnSpinner() {
    jr.jspinner.show({ query: '#demoSpinnerA', relative: true });
}
//---------------------------------
  • jr.jspinner.hide()

    Oculta la animación que bloquea la interfaz.

    Para el siguiente ejemplo se agrega un spinner aleatorio a un elemento div, dentro de las acciones que se tienen para éste ejemplo es: mostrar el spinner y eliminarlo.

    Selecciona un tema!

      

    var btnSpinnerShow = document.querySelector('#btnSpinnerBShow'),
	btnSpinnerFade = document.querySelector('#btnSpinnerBFade');

btnSpinnerShow.addEventListener('click', fnSpinner, false);
btnSpinnerFade.addEventListener('click', fnSpinner, false);

//---------------------------------
// Función para mostrar o bloquear elementos
function fnSpinner() {
    if(this.id == 'btnSpinnerBShow') {
        jr.jspinner.show({ query: '#demoSpinnerB', relative: true });
    } else {
        jr.jspinner.hide({ query: '#demoSpinnerB', relative: true });
    }
} 
//---------------------------------
